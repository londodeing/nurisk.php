# Role-Based Sitemap V2

Pemetaan navigasi menu baru yang secara tegas menghilangkan tumpang-tindih peran fungsional.

## 1. TRC (Mobile Field Role)
- **Menu Utama:** Nearby Incidents (Peta Utama)
- **Modul:** Assessment Form, Quick Report, My Assignment
- **Approval:** (Tidak Ada)

## 2. Operator Posko (Data Entry Role)
- **Menu Utama:** Data Input Center (Dashboard)
- **Modul:** Mutasi Logistik, Absensi Relawan, Sitrep Drafter
- **Approval:** (Tidak Ada)

## 3. Komandan Posko (Tactical Manager Role)
- **Menu Utama:** Posko Command Board
- **Modul:** Resource Status, Decision Queue, Volunteer Assignment
- **Approval:** Eskalasi Posko (Pengiriman/Penerimaan)

## 4. Koordinator Klaster (Sectoral Monitor)
- **Menu Utama:** Cluster Gap Analysis
- **Modul:** Sectoral Needs, Stock Cannibalization
- **Approval:** Pemindahan aset lintas posko (sektoral)

## 5. Ketua PCNU (Branch Executive)
- **Menu Utama:** Mission Coordination Center
- **Modul:** Posko Health Matrix, Escalation Queue
- **Approval:** Approval Center (Surat Cabang, Eskalasi Posko)

## 6. Pimpinan PWNU (Provincial Executive)
- **Menu Utama:** Executive Dashboard
- **Modul:** Provincial Trends, Critical Areas
- **Approval:** Approval Center (Surat Wilayah, Pleno Provinsi)

## 7. Command Center Operator (Monitor Role)
- **Menu Utama:** Live Big Screen (Tanpa Intervensi)
- **Modul:** Auto-Polling Feed, Peta Bencana
- **Approval:** (Tidak Ada)
