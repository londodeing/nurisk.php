# NURISK Phase 19A — Human Efficiency Metrics

Hitungan analitis (*baseline*) mengenai kecepatan dan interaksi operator untuk menyelesaikan satu aksi esensial. Target seluruh aksi di bawah ini adalah maksimal **3 klik**.

## 1. Klik Menuju Aksi Utama (Quick Actions)
- **Target:** 2 Klik.
- **Alur:** Buka Dashboard (1) -> Klik tombol *Quick Action* di Header (2) -> Modal Tampil.

## 2. Klik Menuju Sitrep
- **Target:** 3 Klik.
- **Alur:** Dashboard Posko (1) -> Klik Tombol "+ Buat Sitrep" (2) -> Isi Form Ringkas (0 klik, ketik saja) -> Klik Submit (3).

## 3. Klik Menuju Penugasan Relawan
- **Target:** 3 Klik.
- **Alur:** Dashboard (1) -> Lihat *Decision Queue* -> Klik tombol aksi cepat "Assign" pada baris tabel (2) -> Konfirmasi "Ya, Berangkat" (3).

## 4. Klik Menuju Mutasi Logistik Keluar
- **Target:** 4 Klik (Toleransi ekstra untuk keamanan stok).
- **Alur:** Klik Widget Stok Kritis (1) -> Modal Barang Tampil, Klik "Distribusi Keluar" (2) -> Masukkan kuantitas (0) -> Klik Simpan (3) -> Klik "Setujui" pada *confirm modal* (4).
