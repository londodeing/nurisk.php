# NURISK Phase 19 — Quick Action Framework

Setiap dashboard (PWNU, PCNU, Posko, Relawan) memiliki blok navigasi khusus berupa **Maksimal 5 Tombol Utama** (*Quick Actions*). Tombol ini harus besar, kontras warnanya tinggi, dan idealnya memunculkan *Modal* tanpa perpindahan halaman (*page load*).

## POSKO (Garis Depan)
1. **+ Buat Sitrep** (Modal: Input cepat situasi lapangan)
2. **+ Tambah Relawan** (Modal: Pindai QR Code atau Input Manual)
3. **+ Buat Penugasan** (Modal: Tugaskan evakuasi ke relawan idle)
4. **+ Input Mutasi Logistik** (Modal: Catat barang masuk/keluar harian)
5. **! Hubungi Komando** (Modal: Kirim pesan SOS ke PCNU)

## PCNU (Manajerial)
1. **+ Terbitkan Surat** (Modal: Pilih template surat instruksi/tugas)
2. **+ Panggil Pleno** (Modal: Jadwalkan rapat penentuan status bencana)
3. **+ Minta BKO** (Modal: Eskalasi minta bantuan relawan dari wilayah lain)
4. **+ Buka Posko Baru** (Modal: Aktivasi titik logistik baru)
5. **+ Tutup Operasi** (Halaman konfirmasi penghentian masa darurat)

## PWNU (Eksekutif)
1. **Instruksi Wilayah** (Modal: Broadcast pesan ke seluruh PCNU)
2. **Persetujuan Eskalasi BKO** (Modal: Approve perpindahan relawan lintas PCNU)
3. **Akses Rekap PDF** (Download langsung laporan LHP)
4. **Hubungi Pemerintah** (Menampilkan kontak BPBD/Gubernur)
5. **Rilis Dana Darurat** (Integrasi modul keuangan/NU Care)

## Relawan (Mobile)
1. **[ Lapor Titik Darurat ]** (Tombol MERAH super besar)
2. **[ Kirim Sitrep ]** (Sync offline/online)
3. **[ Selesai Tugas ]** (Tandai penugasan rampung)
4. **[ Absen/Check-in ]** (Rekam titik GPS)
5. **[ Butuh Bantuan ]** (Panic Button)
